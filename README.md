# Most Valuable Store - Data Analysis Portfolio Project
his project analyzes beverage sales data across different stores to identify which products sell best in different locations.

## Objectives
- Identify which products perform best in each store
- Compare sales trends across multiple locations
- Analyze vendor performance (Pepsi vs Coke, etc.)

## Getting Started

Step by step guide on how to use this project
1. Clone this reposition
2. Install dependencies using 'pip install -r requirements.txt'
3. Add your CSV file to 'data/raw'
4. Open Jupyter 'jupyter notebook'
5. Start with 'notebooks/01_data_exploration.ipynb'

## Data Source
Downloaded from Kaggle: 'Beverage Sales Dataset', 'https://www.kaggle.com/datasets/sebastianwillmann/beverage-sales'
File size: 744 MB (Stored locally, not on github, using .gitignore)
